程序名称：坦克大战2020全新重制版
——————————————————————————
程序所用到的所有库：
pygame：用于整个游戏界面的铺设，游戏骨架的搭建
sys：用于系统命令的调用，在这里用于系统层面结束程序
random：用于随机产生奖励buff的位置
math：用于开平方，三角函数等数学运算

——————————————————————————
运行方法：
将文件夹E放到与源代码同一目录下，在解释器中编译运行main.py即可
